# Guns
A collection of practical gun examples.

---

## Examples
* **[Example Bullet](https://github.com/tmodders/example-mod/blob/main/Content/Guns/ExampleBullet.cs):** An example of a simple bullet.
* **[Example Bursts Gun](https://github.com/tmodders/example-mod/blob/main/Content/Guns/ExampleBurstsGun.cs):** An example of a gun that shoots in bursts.
* **[Example Conversion Gun](https://github.com/tmodders/example-mod/blob/main/Content/Guns/ExampleConversionGun.cs):** An example of a gun that converts musket bullets into example bullets.
* **[Example Even Spread Gun](https://github.com/tmodders/example-mod/blob/main/Content/Guns/ExampleEvenSpreadGun.cs):** An example of a gun that shoots bullets in an even spread.
* **[Example Gun](https://github.com/tmodders/example-mod/blob/main/Content/Guns/ExampleGun.cs):** An example of a simple gun.
* **[Example Muzzle Offset Gun](https://github.com/tmodders/example-mod/blob/main/Content/Guns/ExampleMuzzleOffsetGun.cs):** An example of a gun that shoots bullets with a muzzle offset.
* **[Example Random Spread Gun](https://github.com/tmodders/example-mod/blob/main/Content/Guns/ExampleRandomSpreadGun.cs):** An example of a gun that shoots bullets in a random spread.
