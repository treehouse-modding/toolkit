# Introduction

This page lists popular tModLoader mods alongside their license, availability, status and repository.

# Availability

> [!CAUTION]
> Public source code does not automatically grant permission to use, redistribute, or modify a mod's assets. Always read the project's license and respect any additional terms provided by the authors.

| Status | Description |
| --- | --- |
| Open Source | The source code is publicly available at all times. |
| Source Available | The source code is publicly available, but only up until the latest released version. |
